tren git co branch
- branch co nghia la moi truong lam viec khac nhau
- trong branch co cac nhanh chinh de chay CI/CD (main). Code o main la code stable (ko co loi, on dinh nhat)
- bth se day code sang cac nhanh khac, khi on dinh roi moi merge vao main, muon merge code thi phai tao merge request

Command on git

- git status: check xem dang o nhanh nao
- git pull: update code, dong bo nhanh main
- git checkout -b ten_branch // -b la branch: tao branch moi
- git checkout main: ve nhanh main
- git checkout -b feature/bai11: tao branch cho bai 11

// muon merge code tu branch bai 11 vao main thi xem phan dau bai 11
